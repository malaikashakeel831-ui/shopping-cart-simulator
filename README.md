# shopping-cart-simulator
Shopping Cart Simulator

A C++-based Shopping Cart Simulator designed to demonstrate Object-Oriented Programming (OOP) concepts through a real-world e-commerce scenario.

Features

* Product management system
* Add and remove items from cart
* Dynamic price calculation
* Discount application
* Tax calculation
* Itemized receipt generation
* Multi-class architecture
* Runtime polymorphism and inheritance

Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL

OOP Concepts Demonstrated

* Classes and Objects
* Inheritance
* Polymorphism
* Encapsulation
* Abstraction

Project Structure

The system consists of multiple product classes that inherit from a common base class, allowing flexible product handling and runtime behavior. The shopping cart processes products, applies pricing rules, calculates taxes and discounts, and generates a detailed receipt.

Learning Outcomes

This project was developed to strengthen understanding of:

* Object-Oriented Design
* Software Architecture
* Class Relationships
* Runtime Polymorphism
* C++ Programming Fundamentals
